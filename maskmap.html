Mask Map

Design Challenge

What’s the design challenge?

Because of the coronavirus, the face masks are in low storage in every country.This design challenge is started by The F2E

The F2E encouraged UI designers and front end developers to contribute to the mask map which helps people search where are the pharmacies still have face masks.


The request

Show info(address, open time) of pharmacies which sell face masks

See the pharmacies nearby the users and see its face mask storage

Show the face masks purchase rules


Design metric

Low learning curve

Clarity UI

Mobile friendly


Design

Take google map as design style reference because people are familiar with it so it has a low learning curve for them

Use different colors to identify adult and child masks helps users scan info quickly

User different colors to identify pharmacies have mask or not helps users scan info quickly

Show distance between users helps user make decision

Allow users to filter mask categories to save users’ time
