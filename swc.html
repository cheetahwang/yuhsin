Define Problem

  • Too many alerts
    According to iOS Human Interface Guidelines, alerts disrupt the user experience and should only be used in important situations. If there are too many situations use alert, people will not take them seriously.
  • Unclear instructions
    There should be a hint about the limit of the password. I felt frustrated when I input the password and then popuped an alert said I should use the password which contains one uppercase character. When I pressed the OK button at Step4, I thought I finished the registraion progress and tried to log in, then an alerts popuped which said that I need to confirm the email address. I think it should inform me that before I tried to log in.
  • Hard to notice the registration button
    It’s uneasy to find the registration button at a glance, and when I tried to press the registration button, I pressed the forgot password button incautiously.
